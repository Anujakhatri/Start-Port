## ⚙️ Local Development

To run this portfolio locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone [_REPOSITORY_URL]
   cd repo_name
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
