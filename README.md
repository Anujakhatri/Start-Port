## ⚙️ Local Development

To run this portfolio locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone [YOUR_REPOSITORY_URL]
   cd Start-Port
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
